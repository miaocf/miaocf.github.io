    二. 事件输出

    1. 日志信息

      键入adb logcat指令会显示日志信息， 使用ctrl + C用来停止输出

    2. dumpsys

      dumsys命令可以提供非常多的系统信息，可通过adb shell service list来查看dumpsys可查看的信息服务，常用的有

      服务 类名 功能
      activity ActivityManagerService AMS相关信息
      package PackageManagerService PMS相关信息
      windows WindowManagerService WMS相关信息
      input InputManagerService IMS相关信息
      power PowerManagerService PMS相关细信息
      procstats ProcessStateService 进程统计
      battery BatteryService 电池信息
      alarm AlarmManagerService 闹钟信息
      meminfo MemBinder 内存
