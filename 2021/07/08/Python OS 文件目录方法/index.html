Python OS 文件/目录方法

发表于 2021-07-08 更新于 2021-07-31

一. 模块OS

Python的OS模块提供了各类方法来处理文件和目录，常用的方法有以下：

1. os.listdir(path)

2. os.chdir(path)

该方法用于改变当前工作目录到指定的路径。允许访问返回True, 不允许访问返回False

3. os.getcwd()

该方法用于获取当前工作路径

https://www.runoob.com/python/os-file-methods.html

4. os.path.join(path1, path2)

该方法用于拼接目录

二、 文件读写方法

1. open(‘fileName’, mode)

该方法用来打开文件，其中还可以带第三个参数： 指定编码格式 encoding = ‘utf-8’

  • f.read( ) : 一次性读取整个文件数据

  • f.readline( ), 一次读取一行数据

  • f.readlines( ), 一次性读取所有数据，并以List形式返回。

注意： with open(file) as f 的方法，相对于直接f = open(file)更推荐，因为第二种方法如果打开文件失败，则会报错，且报错后不执行f.close()。 因此需要等价使用try… finally 语句，如下：

每次都这么写太麻烦，因此引入with open … as f 语句， 两者等价，且不需要显示调用f.close().

2. f.write(“Text”)

将内容写入文件的方法大致同上， 应用打开的文件变量，直接f.write( )即可。

Tip：python中， 表示路径时， 正斜杠’ \\ ‘需要两条, 反斜杠‘ / ‘只需要一条
