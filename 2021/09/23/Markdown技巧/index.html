Markdown技巧

发表于 2021-09-23 更新于 2022-02-24 分类于 Markdown

简介

本文主要介绍一些markdown编辑格式下的一些技巧。

一. emoji表情

MarkDown支持插入emoji表情，具体方法可以分为两种

  • 方法一：使用Unicode

    格式：&#x + unicode + ;

    Unicode Markdown emoji
    U+1F44D &#x1F44D; &#x1F44D;
    U+1F30D &#x1F30D; &#x1F30D;
    U+1F603 &#x1F603; &#x1F603;

    详细的emoji集合和编码可以参考：[Full Emoji List, v14.0][http://www.unicode.org/emoji/charts/full-emoji-list.html]

    该方法的缺点是需要记住对应的Unicode编码，不实用。

  • 方法二： 使用英文单词

    格式：:+对应英文单词+:

    emoj Markdown格式
    :smile: :smile:
    :bulb: :bulb:
    :mag: :mag:
    :star: :star:

    该方法通过两个冒号和具体的英文单词拼接而成，使用方便。比较实用。

二. 实体键盘按键

格式： 使用<kbd>+按键名+</kbd>

代码效果：Ctrl Tab Anything

三. 特殊格式

名称 格式 效果
删除线 ~~anthing~~ anthing
加粗 **hello world** / Ctrl+T hello world
斜体 *hello world* / Ctrl+B hello world
斜体加粗 ***hello world*** hello world
